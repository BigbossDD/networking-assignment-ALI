# Personal Learning Notes

## Git & GitHub
- Git tracks changes in files using commits
- Branches let you work on features without breaking main code
- Always write meaningful commit messages

## Networking
- IP addresses identify devices on a network
- Subnet masks split the network and host portions
- CIDR makes subnetting easier to read (e.g. /24)

## Subnetting
- Borrowing bits from host portion creates more subnets
- Formula: 2^n subnets, 2^h - 2 usable hosts
- Always assign largest subnet first (VLSM)
